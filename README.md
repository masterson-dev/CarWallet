# Projeto CarWallet 🚗💼

**Disciplina:** Padrões de Projetos de Software
**Avaliação:** 1AV

## 📌 Sobre o Projeto
O CarWallet é um sistema focado na gestão financeira de veículos. O objetivo principal é unificar o registro de despesas financeiras com o acompanhamento de quilometragem (odômetro), permitindo que proprietários de veículos calculem a eficiência (km/l) e gerenciem seus custos mensais de forma rápida e com mínima fricção.
